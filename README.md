



# Mini Company Knowledge Bot

این پروژه یک ربات ساده برای پاسخگویی به سوالات کارمندان بر اساس مستندات داخلی شرکت است.

## 1. ساختار پروژه
- `docs/`: شامل فایل‌های متنی حاوی اطلاعات شرکت.
- `src/`: شامل کدهای اصلی برنامه (bot.py).

## 2. پیش‌نیازها
- Python 3.x

## 3. نحوه اجرا
1. مخزن را کلون کنید.
2. با دستور زیر برنامه را اجرا کنید:
```bash
   python src/bot.py


## 4. دستور های ورودی 

از کلمات مشترک موجود در جملات داکس استفاده کنید
توجه داشته باشید که حداقل باید از دو کلمه ی مشترک استفاده کنید 

برای مشاهده ی نمومه به فایل agentic-bain/EVALS.md مراجعه کنید
